Análise de Vendas no Varejo

Este projeto analisa dados de vendas do varejo com o objetivo de identificar padrões de consumo, desempenho por categoria e comportamento dos clientes.

O pipeline de dados foi desenvolvido no Databricks utilizando SQL e Python, com posterior visualização dos insights em um dashboard.
