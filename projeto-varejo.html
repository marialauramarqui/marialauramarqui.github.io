Análise de Vendas no Varejo

Este projeto tem como objetivo analisar dados de vendas do varejo, explorando padrões de consumo, categorias mais vendidas e comportamento dos clientes.

O pipeline de dados foi desenvolvido no Databricks utilizando SQL e Python, e os resultados foram apresentados em um dashboard.
